# Tic Tac Toe ⭕

A classic Tic Tac Toe game with single and multiplayer modes.

## ✨ Features

- Single player vs AI
- Two player mode
- AI difficulty levels
- Score tracking
- Game history
- Animations
- Sound effects
- Responsive design

## 🛠️ Technical Stack

- HTML5 structure
- CSS3 animations
- Vanilla JavaScript
- Minimax algorithm
- Local Storage
- Responsive design

## 🎮 Game Modes

- Player vs Player
- Player vs AI
- AI vs AI demo
- Tournament mode
- Practice mode
- Custom board size

## 🚀 Getting Started

1. Clone this directory
2. Open `index.html` in your browser
3. Choose mode and play!
