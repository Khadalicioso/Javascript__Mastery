# Speed Type ⌨️

A typing speed test application with various modes and statistics tracking.

## ✨ Features

- Words per minute (WPM)
- Accuracy tracking
- Real-time feedback
- Multiple test modes
- Progress tracking
- Custom text input
- Statistics history
- Performance graphs

## 🛠️ Technical Stack

- HTML5 structure
- CSS3 animations
- Vanilla JavaScript
- Text processing
- Local Storage
- Responsive design

## 📊 Statistics

- WPM calculation
- Accuracy percentage
- Error rate
- Time tracking
- Character count
- Progress over time
- Heat maps
- Personal bests

## 🚀 Getting Started

1. Clone this directory
2. Open `index.html` in your browser
3. Start typing!
