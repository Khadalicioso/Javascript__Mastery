# Ping Pong 🏓

A classic ping pong game implementation with modern features and smooth gameplay.

## ✨ Features

- Single player vs AI
- Two player mode
- Score tracking
- Difficulty levels
- Sound effects
- Power-ups
- Game statistics
- Customizable paddles

## 🛠️ Technical Stack

- HTML5 Canvas
- CSS3 styling
- Vanilla JavaScript
- RequestAnimationFrame
- Web Audio API
- Local Storage
- Responsive design

## 🎮 Game Features

- Smooth paddle movement
- Ball physics
- Collision detection
- Speed progression
- Score system
- Win conditions
- Game pause

## 🚀 Getting Started

1. Clone this directory
2. Open `index.html` in your browser
3. Start playing!

## 🎯 Game Modes

- Practice mode
- AI opponent
- Two player
- Tournament mode
- Time attack

## ⚙️ Controls

- Player 1: W/S keys
- Player 2: Up/Down arrows
- Space: Pause
- R: Reset game
- M: Mute sound

## 🏆 Scoring System

- Point scoring
- Win conditions
- High scores
- Statistics tracking
- Achievement system
