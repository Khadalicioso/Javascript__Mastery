# Gemini Clone 🤖

A clone of Google's Gemini AI chat interface with a modern and responsive design.

## ✨ Features

- Chat interface
- AI response simulation
- Message history
- Code highlighting
- Markdown support
- Dark/Light mode
- Responsive design
- Message threading

## 🛠️ Technical Stack

- HTML5 structure
- CSS3 with modern UI
- Vanilla JavaScript
- Markdown parser
- Code syntax highlighter
- Local Storage
- Responsive design

## 💬 Chat Features

- Real-time responses
- Message formatting
- Code blocks
- Image support
- Voice input
- Chat export
- History management

## 🚀 Getting Started

1. Clone this directory
2. Open `index.html` in your browser
3. Start chatting!

## 🎨 UI Features

- Clean interface
- Smooth animations
- Theme switching
- Mobile optimization
- Accessibility support

## 💾 Data Management

- Chat history
- User preferences
- Local storage
- Export/Import

## ⚙️ Configuration

```javascript
const config = {
    theme: 'light',
    codeHighlight: true,
    saveHistory: true,
    maxHistory: 100
};
```
