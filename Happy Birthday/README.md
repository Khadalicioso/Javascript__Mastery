# Happy Birthday 🎂

An interactive birthday celebration webpage with animations and personalized features.

## ✨ Features

- Customizable birthday messages
- Animated cake
- Floating balloons
- Birthday music
- Confetti effects
- Name customization
- Shareable links
- Responsive design

## 🛠️ Technical Stack

- HTML5 Canvas
- CSS3 Animations
- Vanilla JavaScript
- Web Audio API
- Local Storage
- Responsive design

## 🎨 Animation Features

- Cake animation
- Balloon physics
- Confetti explosion
- Text animations
- Particle effects
- Interactive elements

## 🚀 Getting Started

1. Clone this directory
2. Open `index.html` in your browser
3. Customize and celebrate!

## 🎵 Audio Features

- Birthday song
- Sound effects
- Volume control
- Mute option

## 🔧 Customization

```javascript
const config = {
    name: 'Friend',
    showConfetti: true,
    playMusic: true,
    balloonCount: 20
};
```
