# Tetris 🎮

A classic Tetris game implementation with modern features and smooth controls.

## ✨ Features

- Classic gameplay
- Score system
- Level progression
- Next piece preview
- Hold piece feature
- Ghost piece
- Sound effects
- High scores

## 🛠️ Technical Stack

- HTML5 Canvas
- CSS3 styling
- Vanilla JavaScript
- RequestAnimationFrame
- Web Audio API
- Local Storage
- Responsive design

## 🎮 Game Features

- Piece rotation
- Hard drop
- Soft drop
- Line clearing
- Level system
- Score multipliers
- Wall kicks
- T-spin detection

## 🚀 Getting Started

1. Clone this directory
2. Open `index.html` in your browser
3. Start playing!

## ⌨️ Controls

- Left/Right: Move piece
- Up: Rotate clockwise
- Z: Rotate counter-clockwise
- Down: Soft drop
- Space: Hard drop
- C: Hold piece
- P: Pause game

## 🏆 Scoring System

- Single line: 100
- Double line: 300
- Triple line: 500
- Tetris: 800
- T-spin bonus
- Back-to-back bonus
- Combo bonus
