# Superhero Search 🦸‍♂️

A comprehensive superhero database search application with detailed character information.

## ✨ Features

- Character search
- Detailed profiles
- Power statistics
- Character comparisons
- Favorites system
- Comics appearances
- Team affiliations
- Gallery views

## 🛠️ Technical Stack

- HTML5 structure
- CSS3 Grid/Flexbox
- Vanilla JavaScript
- Superhero API
- Local Storage
- Responsive design

## 🦹 Search Features

- Name search
- Power search
- Publisher filter
- Team filter
- Alignment filter
- Year introduced
- Powers filter

## 🚀 Getting Started

1. Clone this directory
2. Open `index.html` in your browser
3. Start searching heroes!
