# Translator App 🌐

A comprehensive language translation application with multiple features and language support.

## ✨ Features

- Text translation
- Language detection
- Voice input
- Text-to-speech
- Phrase book
- History tracking
- Offline mode
- Multiple languages

## 🛠️ Technical Stack

- HTML5 structure
- CSS3 with modern UI
- Vanilla JavaScript
- Translation API
- Speech API
- Local Storage
- Responsive design

## 🗣️ Translation Features

- Real-time translation
- Auto-language detection
- Pronunciation guide
- Character count
- Copy to clipboard
- Share translations
- Favorites system

## 🚀 Getting Started

1. Clone this directory
2. Open `index.html` in your browser
3. Start translating!

## 🌍 Supported Languages

- Major world languages
- Regional dialects

- Right-to-left support
- Language variants
- Custom language pairs
